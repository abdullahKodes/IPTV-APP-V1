# Antigravity Project Brief

Build and maintain this as a Roku TV IPTV app named IPTV Max.

Follow `AGENTS.md` and keep all work inside this folder. Use Roku SceneGraph and BrightScript/BrighterScript patterns. Do not convert the app into a web app.

Important priorities:

- TV-first navigation with clear remote focus states.
- Reliable HLS/IPTV playback architecture when real streams are added.
- Playlist flows for M3U URL and Xtreme Codes-style accounts.
- Monetization readiness: premium account state, ad planning, privacy policy, content rights, and store assets.
- Never commit provider credentials, Roku device passwords, signing keys, payment keys, or API tokens.
