# IPTV Max Roku App

This is a Roku SceneGraph IPTV app workspace based on the supplied IPTV Max design pages.

## Project Structure

- `source/main.brs` launches the Roku SceneGraph screen.
- `components/MainScene.xml` declares the main SceneGraph component.
- `components/MainScene.brs` builds the IPTV Max UI, navigation, sample data, settings toggles, and screen routing.
- `manifest` contains Roku channel metadata and required libraries.
- `build/` is generated by BrighterScript and is not committed.

## Commands

- `npm.cmd run check` validates and packages the app locally.
- `npm.cmd run build` creates the Roku zip.
- `npm.cmd run deploy` deploys to a configured Roku device.

## Current Screens

- Home dashboard
- Add Playlist with M3U and Xtreme account modes
- My Playlists manager
- Live TV with channel list, player panel, and EPG strip
- Series browser
- Movies browser
- Settings
- Favourites
- Notifications

Real IPTV provider credentials, stream URLs, payment provider keys, and signing credentials must stay outside source control.
